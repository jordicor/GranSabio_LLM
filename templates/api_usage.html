Quick Start Guide

Get started with Gran Sabio LLM in three simple steps:

1

Submit Request

Send your content requirements to the /generate endpoint

2

Monitor Progress

Track real-time updates via /stream/project/{project_id}

3

Retrieve Content

Get your quality-assured content from /result/{session_id}

Simple Example

Python 
import requests
import json

# Step 1: Generate content
response = requests.post('https://api.example.com/generate', json={
    "prompt": "Write a professional biography of Marie Curie",
    "content_type": "biography",
    "max_words": 1000,
    "qa_layers": ["accuracy", "readability", "engagement"]
})

session_id = response.json()['session_id']

# Step 2: Check status
status = requests.get(f'https://api.example.com/status/{session_id}')
print(f"Status: {status.json()['status']}")

# Step 3: Get result (when completed)
result = requests.get(f'https://api.example.com/result/{session_id}')
content = result.json()['content']

Authentication

Authenticate your requests using your API key in the header:

HTTP Header

Real-time Streaming

GET /stream/project/{project_id}

Stream live progress updates using Server-Sent Events. When no explicit project_id is provided in the generation request, the session_id is used as the project_id.

JavaScript 
const eventSource = new EventSource(`/stream/project/${projectId}`);

eventSource.onmessage = (event) => {
  const update = JSON.parse(event.data);
  console.log(`Progress: ${update.progress}%`);
  console.log(`Status: ${update.status}`);

  if (update.status === 'completed') {
    eventSource.close();
    // Retrieve the final result
  }
};

Complete Examples

Example 1: Professional Biography

Python 
import requests
import time

def generate_biography(subject_name, focus_areas):
    # Submit generation request
    response = requests.post(
        'https://api.example.com/generate',
        headers={'Authorization': f'Bearer {API_KEY}'},
        json={
            "prompt": f"Write a comprehensive biography of {subject_name}, "
                     f"focusing on: {', '.join(focus_areas)}",
            "content_type": "biography",
            "generator_model": "claude-sonnet-4-20250514",
            "min_words": 1500,
            "max_words": 2000,
            "temperature": 0.7,
            "qa_layers": [
                {
                    "name": "Factual Accuracy",
                    "criteria": "All facts, dates, and events must be verifiable",
                    "min_score": 9.0
                },
                {
                    "name": "Engagement",
                    "criteria": "Compelling narrative that maintains reader interest",
                    "min_score": 8.0
                }
            ]
        }
    )

    session_id = response.json()['session_id']

    # Poll for completion
    while True:
        status = requests.get(
            f'https://api.example.com/status/{session_id}',
            headers={'Authorization': f'Bearer {API_KEY}'}
        ).json()

        if status['status'] == 'completed':
            break
        elif status['status'] == 'failed':
            raise Exception(f"Generation failed: {status.get('error')}")

        time.sleep(5)

    # Get final result
    result = requests.get(
        f'https://api.example.com/result/{session_id}',
        headers={'Authorization': f'Bearer {API_KEY}'}
    ).json()

    return result['content']

# Use the function
biography = generate_biography(
    "Marie Curie",
    ["scientific discoveries", "personal challenges", "legacy"]
)
print(biography)

Example 2: Technical Article with Research

JavaScript 
async function generateTechnicalArticle(topic, researchDocs) {
    // First, upload research documents
    const uploads = await Promise.all(
        researchDocs.map(doc => uploadDocument(doc))
    );

    // Generate article with context
    const response = await fetch('https://api.example.com/generate', {
        method: 'POST',
        headers: {
            'Authorization': `Bearer ${API_KEY}`,
            'Content-Type': 'application/json'
        },
        body: JSON.stringify({
            prompt: `Write a technical article about ${topic}`,
            content_type: "article",
            generator_model: "gpt-5",
            max_words: 3000,
            username: "tech_writer",
            context_documents: uploads.map(u => ({
                upload_id: u.id,
                intended_usage: "Research reference"
            })),
            qa_layers: [
                {
                    name: "Technical Accuracy",
                    criteria: "All technical concepts must be correct",
                    min_score: 9.0
                },
                {
                    name: "Clarity",
                    criteria: "Complex concepts explained clearly",
                    min_score: 8.5
                }
            ]
        })
    });

    const { session_id } = await response.json();

    // Stream progress (project_id = session_id when not explicitly provided)
    const eventSource = new EventSource(
        `https://api.example.com/stream/project/${session_id}`
    );

    return new Promise((resolve, reject) => {
        eventSource.onmessage = (event) => {
            const data = JSON.parse(event.data);
            console.log(`Progress: ${data.progress}%`);

            if (data.status === 'completed') {
                eventSource.close();
                fetch(`https://api.example.com/result/${session_id}`)
                    .then(r => r.json())
                    .then(result => resolve(result.content));
            }
        };
    });
}

Troubleshooting

Generation Taking Too Long

Possible Causes:

  • High quality standards requiring multiple refinements
  • Complex content requiring deep analysis
  • Using reasoning models which require more processing time

Solutions:

  • Adjust quality thresholds if appropriate
  • Use faster models for initial drafts
  • Break complex content into smaller sections

Content Not Meeting Quality Standards

Solutions:

  • Provide more detailed prompts
  • Include reference materials via attachments
  • Adjust quality layer criteria to be more specific
  • Use premium models for critical content

Session Not Found

Causes:

  • Session expired (sessions are kept for 24 hours)
  • Invalid session ID

Solution: Store session IDs immediately after generation request and retrieve results promptly.
